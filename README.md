# Rhythm-game
---

## **프로젝트 개요**

### **설명**

> Pygame을 활용한 리듬게임 제작.<br><br>
> 장애물을 피해서 마지막 지점까지 달려가기.<br><br>
> 마지막 지점까지 달리면 클리어 타임과 점수가 출력.<br><br>
> 재생되는 리듬에 맞게 Space를 입력, 남은 체력, <br>
  생존한 시간에비례하여 점수 추가.

### **사용기술**

> Python, Pygame, Github
